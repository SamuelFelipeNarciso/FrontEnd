Comandos de C

Aqui você verá alguns comandos de C

  • strlen
  • strcpy
  • strcat
  • strcmp
  • strchr
  • strcspn
  • printf
  • scanf
  • fgets
  • if-else
  • for
  • while
  • do-while

strlen

Retorna o tamanho da string (quantidade de letras), sem contar o caractere final \0.

int tam = strlen("Samuel"); // O resultado é 6

strcpy

Copia o conteúdo de uma string para outra (origem para destino).

strcpy(destino, "Ola"); // A variável destino recebe o texto "Ola"

strcat

Junta (concatena) duas strings, colocando a segunda ao final da primeira.

strcat(txt, " Mundo"); // Se txt era "Ola", vira "Ola Mundo"

strcmp

Compara duas strings. Se elas forem exatamente iguais, o resultado é 0.

if (strcmp(s1, s2) == 0) { // Indica que são iguais }

strchr

Localiza a primeira ocorrência de um caractere específico em uma string.

char *posicao = strchr(nome, 'a'); // Procura a letra 'a' dentro de nome

strcspn

É muito usado para remover o \n (Enter) que o comando fgets deixa no final da string.

nome[strcspn(nome, "\n")] = '\0'; // Localiza o \n e troca pelo fim da string \0

printf

Mostra informações na tela.

printf("Olá mundo!");

scanf

Lê dados formatados do teclado (como números inteiros ou decimais).

scanf("%d", &idade); // O & é necessário para variáveis simples

fgets

Lê uma string do teclado de forma segura.

fgets(nome, 50, stdin);

if / else

Estrutura de decisão: executa um bloco se a condição for verdadeira, ou outro se for falsa.

if(idade >= 18) { printf("Maior"); } else { printf("Menor"); }

for

Estrutura de repetição usada quando se sabe o número de vezes que vai repetir.

for(int i=0; i < 10; i++) { printf("%d", i); }

while

Repete um bloco enquanto uma condição for verdadeira.

while(condicao == 1) { // faz algo }

do while

Semelhante ao while, mas garante que o código seja executado pelo menos uma vez.

do { // executa primeiro } while(condicao);
