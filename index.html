Comandos de String em C

Aqui você verá alguns comandos de string para usar em C

strlen

Retorna o tamanho da string (quantidade de letras), sem contar o caractere final \0.

int tam = strlen("Samuel"); // O resultado é 6

strcpy

Copia o conteúdo de uma string para outra (origem para destino).

strcpy(destino, "Ola"); // A variável destino recebe o texto "Ola"

strcat

Junta (concatena) duas strings, colocando a segunda ao final da primeira.

strcat(txt, " Mundo"); // Se txt era "Ola", vira "Ola Mundo"

strcmp

Compara duas strings. Se elas forem exatamente iguais, o resultado é 0.

if (strcmp(s1, s2) == 0) { // Indica que são iguais }

strchr

Localiza a primeira ocorrência de um caractere específico em uma string.

char *posicao = strchr(nome, 'a'); // Procura a letra 'a' dentro de nome

strcspn

É muito usado para remover o \n (Enter) que o comando fgets deixa no final da string.

nome[strcspn(nome, "\n")] = '\0'; // Localiza o \n e troca pelo fim da string \0
